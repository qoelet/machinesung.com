Setting up an Alpine box for development


# First bootup, login as `root`, no password
# Follow instructions along, `sys` for a traditional install
$ setup-alpine

# After reboot, let's add a user and update packages
$ adduser foo
$ apk update
$ apk upgrade
$ apk add sudo
$ visudo # uncomment %wheel
$ addgroup foo wheel
$ exit

# Login as user, install libraries
$ sudo apk add alpine-sdk
$ gcc hello.c -o hello
$ ./hello
Hello, Alpine!
