A growing list of my favorite CLI tools

  • Taskwarrior: A great way for tracking your todos.
  • hledger: I use this to track all my expenses.
  • entr: I use this for reloading tests during TDD.
  • gnu-pw-mgr: Generating secure passwords for services that I don't have to remember.
  • cmus: A really nice console music player. (for tuning into SomaFM)
  • frink: A useful way to calculate things.
  • nnn: Fast commandline file manager.
  • keynav: An interesting replacement for using a mouse. I use this one my Odroid XU4; you need to spend some time getting used to it but it does the job!
